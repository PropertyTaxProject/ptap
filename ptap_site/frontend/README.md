# PTAP Frontend

## Directory Layout
```
src
├── admin - Holds components regarding the admin dashboard
├── appeal - Holds components regarding the appeal process
│   ├── appeal.jsx - Appeal form
│   ├── comparables - subdirectory with comparables related components
│   │   └── characteristics.jsx
│   ├── damage - subdirectory with damage input related components
│   │   ├── damage-input.jsx
│   │   ├── damages.jsx
│   │   └── image-upload.jsx
│   └── homeowner - homeowner info entry components
│       ├── form-input.jsx
│       └── pin-lookup.jsx
├── general - general components used across all pages
│   └── header.jsx
├── index.css
├── index.jsx - index file
├── landing - landing page for switching between applications
│   └── landing-page.jsx
├── logo.svg
├── requests.js
├── serviceWorker.js
└── setupTests.js
```
